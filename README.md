# Word 关键词高亮智能工具 (light_docx)

这是一个用于自动在 Word 文档 (`.docx`) 中标记和高亮关键词的 Python 桌面工具及脚本。支持基于段落和表格的关键词匹配，提供三种匹配规则（模糊包含、绝对匹配、排除匹配），并支持图形界面 (GUI) 和命令行接口 (CLI) 双重操作模式。

---

## ✨ 主要功能

1. **多规则匹配**：
   - **高亮关键词**：包含匹配（只标记词本身，支持英文包含匹配）。
   - **特殊关键词**：绝对精确匹配（全词匹配，避免匹配部分单词）。
   - **排除关键词**：排除匹配（如果高亮的部分位于被排除短语内部，则不进行高亮）。
2. **段落与表格高亮**：完整支持 Word 文档中的正文段落及表格内段落的高亮标注。
3. **自动初始化**：软件首次运行（无论 GUI 还是 CLI）会自动在同目录下创建 `关键词配置` 文件夹，并生成上述三个空的文本配置文件，用户直接填入关键词即可使用。
4. **多种高亮颜色**：GUI 界面提供黄色、绿色、青色、粉色、红色、蓝色等多种颜色供选择。
5. **大小写开关**：默认不区分大小写；开启后只匹配大小写完全一致的内容。
6. **跨平台支持**：支持 Windows 和 macOS，并分别提供 Intel / Apple Silicon macOS 打包配置。

## 🛠️ 环境准备与安装

运行本程序需要安装 Python 3 环境。请在终端/命令行窗口中执行以下命令安装依赖库：

```bash
pip install python-docx regex customtkinter
```

*如果在中国大陆地区，可以使用清华镜像加速安装：*
```bash
pip install python-docx regex customtkinter -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 🚀 使用指南

### 1. 自动生成配置文件
直接启动 GUI 或 CLI，程序会在当前路径下自动创建一个 `关键词配置` 文件夹，并在该文件夹下生成以下三个 `.txt` 文件：
- **`高亮关键词.txt`**：用于放置需要高亮的关键词。如果是中文，则进行包含匹配；如果是英文单词，则会匹配包含它的内容。
- **`特殊关键词.txt`**：用于进行绝对/全词匹配。例如，希望仅在英文单词独立出现时高亮（如只匹配 `apple` 而不匹配 `apples`），请将其放入此文件。
- **`排除关键词.txt`**：用于防止误标记。例如，希望高亮 `AI`，但当出现 `AI Agent` 时不希望高亮其中的 `AI`。此时可以在 `排除关键词.txt` 中填入 `AI Agent`。

### 2. 方式一：图形界面运行 (GUI)
双击运行 `高亮工具GUI.py` 或在命令行中运行：
```bash
python 高亮工具GUI.py
```
- **操作步骤**：
  1. 点击“选择文件”按钮，选取您需要高亮的 `.docx` 格式文档。
  2. 默认会自动读取同级目录下的 `关键词配置` 文件夹。如需自定义，可点击“选择文件夹”手动选取存放 TXT 配置文件的目录。
  3. 在下拉菜单中选择您喜欢的高亮颜色（如黄色、绿色等）。
  4. 如需严格匹配英文大小写，打开“区分大小写”开关；默认关闭时不区分大小写。
  5. 点击“开始高亮处理”按钮，程序将自动在后台进行识别、替换与合并处理。
  6. 处理成功后，会在原 Word 文档同一目录下生成一个带有 `[原文件名]高亮.docx` 后缀的新文档。

### 3. 方式二：命令行脚本运行 (CLI)
打开 `高亮关键词.py`，在文件顶部配置以下参数：
```python
WORD_DOCUMENT_PATH = r'你的Word文档路径.docx'
OUTPUT_DOCUMENT_PATH = r'保存高亮结果文档路径.docx'
CASE_SENSITIVE_MATCH = False  # True=区分大小写；False=不区分大小写
```
然后在命令行中直接运行该脚本：
```bash
python 高亮关键词.py
```

## 📦 打包为独立可执行文件 (无需安装 Python)

### Windows 打包 (生成 `.exe`)
安装 `pyinstaller`：
```bash
pip install pyinstaller
```
在项目目录下执行：
```bash
pyinstaller --windowed --onefile --name="Word高亮工具" 高亮工具GUI.py
```
打包成功后，可在 `dist/` 文件夹下找到 `Word高亮工具.exe`。

### macOS 打包 (生成 `.app` / `.dmg`)
请参考项目目录下的 [macOS打包步骤.md](file:///C:/Users/newnew/Desktop/高亮标记/macOS%E6%89%93%E5%8C%85%E6%AD%A5%E9%AA%A4.md) 获取详细的步骤和 Gatekeeper 安全拦截解锁指南。

- `Word高亮工具_macOS_AppleSilicon.zip`：M 系列芯片 Mac 使用。
