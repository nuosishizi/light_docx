# macOS (苹果系统) 软件打包指引

本指南将教你如何在 Mac 电脑上，将编写好的 [高亮工具GUI.py](file:///c:/Users/newnew/Desktop/高亮标记/高亮工具GUI.py) 打包成一个无需安装 Python 环境即可独立运行的苹果桌面应用（`.app` 或 `.dmg`）。

> [!IMPORTANT]
> **关于系统限制的提醒**
> 苹果的软件打包工具（如 `PyInstaller`）**不支持跨平台打包**。你必须在一台 **Mac 电脑**（苹果电脑）上运行以下步骤，无法直接在 Windows 上生成 Mac 格式的 `.app`。

---

## 第一步：Mac 上的环境准备

1. **安装 Python 3**：
   * 打开 Mac 上的“终端”（Terminal）应用（可以通过按下 `Cmd + 空格键` 搜索“终端”打开）。
   * 输入命令检查是否已安装 Python：
     ```bash
     python3 --version
     ```
   * 如果提示找不到命令，请前往 [Python 官方下载页面](https://www.python.org/downloads/) 下载适用于 macOS 的最新安装包（如 Python 3.10 或更高版本）并双击安装。

2. **安装程序依赖包**：
   * 在终端中，运行以下命令安装运行和界面所需的全部依赖：
     ```bash
     pip3 install python-docx regex customtkinter pyinstaller
     ```
   * *提示：如果在安装过程中提示权限不足，可在命令前加上 `sudo`，或者运行 `pip3 install --user ...`*。

---

## 第二步：在 Mac 上测试运行

在正式打包前，建议先测试脚本是否能在 Mac 上正常拉起界面：

1. 将 Windows 上的代码文件夹复制到 Mac 电脑的某个目录（例如“下载”或“桌面”）。
2. 在终端中使用 `cd` 命令进入该文件夹目录。例如如果文件夹放在桌面：
   ```bash
   cd ~/Desktop/高亮标记
   ```
3. 运行程序进行测试：
   ```bash
   python3 高亮工具GUI.py
   ```
   如果窗口成功弹出，且能正常选择文档、读取关键词和运行高亮逻辑，说明环境一切正常！

---

## 第三步：使用 PyInstaller 进行打包

在终端处于代码目录的前提下，执行以下打包命令：

```bash
pyinstaller --windowed --onefile --name="Word高亮工具" 高亮工具GUI.py
```

### 选项参数解析：
* `--windowed` (或 `-w`)：指定其为窗口图形程序，双击运行时**不会**弹出一个黑色的命令行黑框。
* `--onefile` (或 `-F`)：将所有的 Python 代码、依赖库、图片图标等全部打包进一个单独的 `.app` 应用程序包中。
* `--name="Word高亮工具"`：指定最终生成的 Mac 应用程序名称。

### 打包结果：
* 编译过程大约需要 1-3 分钟。
* 编译完成后，你会看到当前目录下多出了几个文件夹。
* **最终生成的 Mac 软件存放在 `dist/` 文件夹中**，名为 **`Word高亮工具.app`**。你可以直接把这个文件拖拽到 Mac 的“应用程序”（Applications）中，或发送给其他 Mac 用户。

---

## 第四步：如何制作 .dmg 安装包（可选）

如果你想向用户分发更加精美的 `.dmg` 磁盘映像格式，可以使用 Mac 自带的“磁盘工具”快速制作，无需安装额外软件：

1. 在 Mac 上打开 **“磁盘工具”**（Disk Utility，可在启动台中找到）。
2. 在顶部菜单栏中选择 **“文件” -> “新建映像” -> “来自文件夹的映像...”**。
3. 选择刚刚打包出来的 `dist` 文件夹（或者包含 `Word高亮工具.app` 的任何文件夹），点击“选取”。
4. 在弹出的保存窗口中：
   * **存储为**：输入最终 dmg 的名称（如 `Word高亮工具安装包`）。
   * **映像格式**：选择 **“只读”**。
5. 点击“存储”即可生成一个标准的 `.dmg` 磁盘文件。

---

## 第五步：如何解决 macOS 安全拦截 (Gatekeeper)

由于我们打包的软件没有向苹果官方申请付费的开发者证书，其他 Mac 用户双击运行你的软件时，系统出于保护机制会弹出安全拦截提示（如“已损坏，无法打开”或“无法确认开发者”）。

请告知其他用户使用以下两种简单方法之一来解除限制：

### 方法一（最简单，通过系统设置解锁）：
1. 第一次双击软件弹出警告后，点击“取消”。
2. 打开 Mac 的 **“系统设置”**，滑到左侧的 **“隐私与安全性”**。
3. 往下拉，会看到一条关于“Word高亮工具 被阻止”的提示，点击其下方的 **“仍要打开”**。
4. 输入 Mac 的锁屏密码确认即可，以后再次打开便不会再弹窗。

### 方法二（通过终端命令永久解除）：
如果你希望将软件打包发给别人，且对方会使用终端，可以让对方在终端运行以下命令：
```bash
xattr -cr /path/to/Word高亮工具.app
```
*(注：只需将 `/path/to/Word高亮工具.app` 替换为软件在 Mac 上的实际路径。也可以先在终端输入 `xattr -cr `，然后直接把 `Word高亮工具.app` 拖入终端窗口，它会自动填充路径，回车即可。)*
